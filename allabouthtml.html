What is HTML

Hyper text markup language is a markup language that uses markup tags to tell the computer what to do. HTML was created byTim Berners-Lee.

How does HTML work?

Let's talk about how HTML works. HTML uses markup language. Unlike programming language, markup languages use markup elements, which describe to a computer what to do.
Markup elements are just text with 2 angle brackets. An example is when you're writing your title. You will write title inside 2 angle brackets, put in your title, and close with /title
inside 2 angle brackets. Same for a paragraph, except you use p instead of title.

Main structure of HTML

In html, you always start with a main structure. That structure is always the same. It will always start with your html tag, which stores the other three parts, head,title, and body, and you can
add more tags in those, such as style.

About the different parts

In HTML, there are 3 tags that you need. Head, which stores title, and body, which stores most of the webpage.

Head tag

The head tag relly only stores te title of the webpage.

Title tag

The title tag makes the the title for a tab in any web browser. It is the text on the tab icon.

Body tag

The body part shows the part of code that is what you see on a screen in a web browser. It is the part that has all of the text.

Additional tags

Also, apart from the main tags, you can add more tags such as style which alllow you to customize your webpage.
